# Twitter Sentiment Analysis
## Problem Statement
This project analyzes tweets to classify sentiment as positive, negative, or neutral.
## Dataset
- Source: Kaggle
- Size: 50,000 tweets
## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
## Models Used
- Logistic Regression
- Naive Bayes
## Results
- Best Accuracy: 87%
## How to Run
1. Install required libraries
2. Run Jupyter Notebook
3. Train the model

## Future Work
